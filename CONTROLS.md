# Controls

This document lists the current key bindings for `Rogue`.

## Main Menu

- `1` new game
- `2` continue (if a save exists)
- `3` statistics
- `4` test mode
- `Q` quit game

## 2D Mode

Movement:
- `W` / `A` / `S` / `D`
- Arrow keys `↑` `↓` `←` `→`

Other:
- `Tab` switch `2D` / `3D`
- `H` use/equip weapon
- `J` use food
- `K` use elixir
- `E` use scroll
- `Q` quit

## 3D Mode

Movement:
- `W` or `↑` move forward
- `S` or `↓` move backward
- `A` or `←` rotate left
- `D` or `→` rotate right

Interaction and combat:
- `F` or `Space` interact (context-sensitive)
- `X` attack
- `G` pick up item

Items:
- `H` weapon
- `J` food
- `K` elixir
- `E` scroll

Visual toggles:
- `Tab` switch `2D` / `3D`
- `T` textures
- `M` mini-map
- `N` sprites
- `I` debug info
- `?` or `/` help
- `Q` or `Esc` quit

## Item Selection Screen

- `1..9` select item by number
- `0` special option (if available, for example `Empty hands`)
- `Q` or `Esc` cancel

## Test Mode Menu

Navigation:
- `↑` / `↓` select option
- `←` or `-` decrease level value
- `→` or `+` / `=` increase level value
- `Space` toggle `Fog of War`
- `Enter` confirm
- `Q` or `Esc` cancel

## Game Over / Victory Screen

- `R` restart
- `M` main menu
- `Q` quit
